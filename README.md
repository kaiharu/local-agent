This is a Local AI Agent with Azure OpenAI LLM Model.

## Getting Started

### Install dependencies

```bash
bun install
```

### Setup .env

```bash
cp .env.example .env.local
```

Plese edit .env.local

### Run app

```bash
bun run dev
```

Open [http://localhost:3123](http://localhost:3123) with your browser to see the result.
